## About Me

I'm an AI Engineer focused on building agentic systems and ML infrastructure that are production-ready — not just demos. My work sits at the intersection of LLM orchestration, retrieval systems, and quantitative modeling.

- Building multi-round **ReAct agents** over legal corpora with LangGraph + vector search
- Designing **loss-based data selection** pipelines that improve LLM fine-tuning by 34.2% on MATH-500
- Running **financial ML systems** with walk-forward backtesting, TCN models, and FastAPI serving
- Obsessed with making AI systems that are measurable, debuggable, and deployable
